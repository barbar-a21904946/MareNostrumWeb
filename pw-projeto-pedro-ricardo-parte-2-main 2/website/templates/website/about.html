Técnicas Detalhes
Tabela de Técnicas Usadas
HTML O HTML foi usado como base deste projeto, desde o começo em um site estático para agora em Django. Serviu também para uma melhor clareza e organização do código.
CSS O CSS é o responsável pela estilização e responsividade do site, desde animações, cores ao estilo do texto.
Django O Django foi utilizado como web framework do site, serviu para passar de um site estático para um dinâmico, renderizando as páginas com dados da BD.
JavaScript O JavaScript serviu forma de ligação entre o DOM e o Django, atualizando as páginas em tempo real.

Django O django foi usado para guardar os dados obtidos nos formulários de quizz, comentários e contacto, estando estes associados a uma class pessoa fazendo assim uso da foreign key. (ver descrição mais detalhada no github)

GitHub O github foi utilizado como local de repositório do projeto, servindo assim para um melhor trabalho de equipa, tendo sempre o projeto atualizado.
